# Ejercicio N° 12
## Desarrollo de Software
### Programación asincrónica

El siguiente proyecto recupera datos de productos de diferentes orígenes (un archivo y un servicio web), obtiene también la cotización del dólar estadounidense de tres servicios distintos y actualiza (con el promedio de las cotizaciones) el precio de los productos.


## Tarea
Mejorar el rendimiento de la aplicación

### Características de la Solución

- Lenguaje: C# 12.0
- Plataforma: .NET 8
